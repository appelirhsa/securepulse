OTP Unlock System

The One-Time Password (OTP) system provides secure unlocking, pairing, and emergency verification for your bracelet.

How It Works

  1. User clicks “Unlock Bracelet” in the app or web portal.
  2. Prompt appears: “Enter OTP to continue”.
  3. OTP is sent to your primary phone number and email (if set).
  4. User enters the OTP code within 60 seconds.
  5. If verified, the bracelet unlocks. If failed, emergency contact is alerted.
Security Note: OTPs are encrypted and time-limited. Only you and your verified contacts can access unlocking options.
